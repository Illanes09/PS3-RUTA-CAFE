# APK — Ruta del Café

| Archivo | Descripción |
|---------|-------------|
| `RutaDelCafe-1.0.0.apk` | App Android (API: Render en la nube) |

## Instalación

1. Descarga `RutaDelCafe-1.0.0.apk` en tu celular.
2. Permite **instalar apps de origen desconocido**.
3. Abre el APK e instala.

## Regenerar el APK

En GitHub: **Actions** → **Build Android APK** → **Run workflow**.

O tras un `git push` que toque `AppMovil_RutaDelCafe/`.
